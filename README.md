## 项目介绍
机械臂，舵机控制相关研究。内含一个机器视觉运动控制项目，使用摄像头捕获特定的人脸目标，然后控制机械臂追随目标移动并定时拍下照片。

## 项目优势
- 了解arduino舵机控制相关编程
- 人脸识别和物体检测相关研究
